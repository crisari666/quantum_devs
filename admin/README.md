# Admin dashboard

Vite + React + TypeScript + Redux Toolkit + Tailwind + React Hook Form + Zod.

## Setup

1. Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` to the NestJS API base (for example `http://localhost:3000/api`).
2. `npm install`
3. `npm run dev`

Sign in with the seeded administrator from the backend `.env` (`ADMIN_SEED_EMAIL` / `ADMIN_SEED_PASSWORD`).

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint
